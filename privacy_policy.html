Privacy Policy for Anime Image Finder

Effective Date: June 29, 2025

Key Point: This is a local desktop application that does not collect, store, or transmit any personal data.

Overview

Anime Image Finder is a local desktop application that helps users find similar anime images on Pinterest. This privacy policy explains how we handle your data.

Information We Collect

We do not collect any personal information.

  • No user accounts or profiles
  • No personal data storage
  • No tracking or analytics
  • No cookies or local storage beyond application functionality

How We Process Images

  • Local Processing Only: All image analysis happens on your computer
  • No Upload: Images are never uploaded to our servers
  • Temporary Processing: Images are processed in memory and discarded
  • Feature Extraction: We extract visual features (colors, edges, patterns) for comparison

Data Usage

  • Pinterest Integration: We only send search queries to Pinterest's public API
  • No Image Transmission: Image data is never sent to external servers
  • Local Storage: Results are saved locally on your computer only
  • API Compliance: We follow Pinterest's API terms of service

Data Sharing

  • No Third-Party Sharing: We do not share any data with third parties
  • Pinterest API: Only authorized API calls to Pinterest for image search
  • Local Files: Generated reports are saved locally on your device

Security

  • Local Execution: All processing occurs on your local machine
  • No Network Transmission: Image data never leaves your computer
  • Secure API Calls: Only encrypted HTTPS requests to Pinterest API
  • No Persistent Storage: No personal data is stored permanently

Your Rights

  • Full Control: You control all data on your local machine
  • No Tracking: We cannot track your usage or behavior
  • Immediate Deletion: Delete the application to remove all local data
  • Transparency: All code is open and inspectable

Contact Information

This is a local desktop application with no data collection or transmission capabilities. For questions about Pinterest's data handling, please refer to Pinterest's privacy policy.

Changes to This Policy

As a local application, we do not have the ability to update this policy remotely. Any changes would require you to download an updated version of the application.

Compliance

This application complies with:

  • GDPR (General Data Protection Regulation)
  • CCPA (California Consumer Privacy Act)
  • Pinterest API Terms of Service
  • Local privacy laws and regulations

This privacy policy reflects our commitment to user privacy and data protection.
